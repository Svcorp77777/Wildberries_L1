Чем отличаются буферизированные и не буферизированные каналы?

Ответ:
```
Не буферизированные каналы - их емкость равна 0, отправитель и получатель 
будут блокироваться во время отправки и получения сообщения, полезны для 
синхронизации горутин.
 
Буферизированные каналы - их емкость больше 0, что позволяет отправителю 
отправлять в канал сообщения пока для этого есть место, при заполнении 
канала отправитель блокируется пока сообщение не будет получено другим 
потоком, полезно когда нужно временно сохранить данные или когда есть 
несколько отправителей или получателей для работы асинхронной схемы.
```
