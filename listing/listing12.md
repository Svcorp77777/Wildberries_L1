Что выведет данная программа и почему?

```go
func main() {
  n := 0
  if true {
     n := 1
     n++
  }
  fmt.Println(n)
}
```

Ответ:
```
Программа выведет 0 так как в if происходит повторная инициализация n 
что приводит к выделению нового адреса в памяти и изменения не выходят 
за пределы if.
```
