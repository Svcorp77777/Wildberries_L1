Что выведет данная программа и почему?

```go
func someAction(v []int8, b int8) {
  v[0] = 100
  v = append(v, b)
}

func main() {
  var a = []int8{1, 2, 3, 4, 5}
  someAction(a, 6)
  fmt.Println(a)
}
```

Ответ:
```
Программа выведет [100 2 3 4 5] так как действие v[0] = 100 изменит 0 элемент 
в изначальном слайсе потому что v хоть и представляет собой копию но также 
указывает на базовый слайс, действие v = append(v, b) не повлияет на основной 
слайс так как создает новую копию с другим адресом в памяти.
```
